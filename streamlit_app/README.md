# 📊 Telco Churn Analysis — Streamlit Dashboard

**Framework:** SCQR (Situation → Complication → Question → Resolution) + Pyramid Principle

Every page follows the same narrative structure:
1. **Situation** — context and background
2. **Complication** — the problem or tension
3. **Question** — what we need to answer
4. **Resolution** — the answer (= Pyramid Level 1)
5. **Level 2** — supporting arguments (MECE)
6. **Level 3** — data evidence (charts, tables, statistical tests)

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```
